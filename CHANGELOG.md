# Changelog

## [v1.5.0] - 2026-04-03
### ✨ Features
- Добавлен `nano1602.service` для автоматического запуска при старте системы.
- Инструкция по настройке systemd в `README.md`.
- Улучшена стабильность работы в фоне.

## [v1.4.1] - 2026-04-03
### ✨ Features
- Первый релиз монитора PC для Arduino Nano.
- Отображение CPU и RAM на LCD 1602 (I2C) с живыми шкалами.
- Автоматическая установка Python-зависимостей через `start.sh`.
- Поддержка работы без `psutil` (через чтение `/proc` для максимальной совместимости).
- Авто-поиск USB-порта Arduino.
