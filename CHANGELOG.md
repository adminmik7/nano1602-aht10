# Changelog

## [v1.7.0] - 2026-04-07
### ✨ Features
- **Visual Upgrade:** Полностью переработан `start.sh`. Теперь он встречает красивым баннером, цветными статусами и чистым выводом.
- **Reliability:** Финальная полировка логики переподключения USB. Скрипт и прошивка теперь работают как единый надежный механизм.

## [v1.6.0] - 2026-04-07
### 🛠 Reliability Fixes
- **USB Reconnect (Python):** Скрипт `sender.py` больше не падает при отключении USB. Реализован автоматический поиск порта и переподключение при ошибке `OSError: [Errno 5]`.
- **USB Status (Arduino):** Добавлена визуальная индикация потери связи. Если данные не приходят 3 секунды, экран очищается и пишет "USB Lost!", а при восстановлении выводит "USB Connected!".
- **Timeout:** Уменьшен таймаут потери связи до 3 секунд для более быстрой реакции.

## [v1.5.0] - 2026-04-03
### ✨ Features
- **Auto-start:** Добавлен `nano1602.service` для запуска монитора при старте системы (systemd).
- **Docs:** Подробная инструкция по настройке автозапуска в `README.md`.
- **Stability:** Улучшена обработка перезапусков сервиса.

## [v1.4.1] - 2026-04-03
### 🛠 Fixes
- **Port Detection:** Улучшено автоопределение USB-порта (проверка на доступность перед подключением).
- **Docs:** Обновлен `README` с полной инструкцией по установке и запуску.

## [v1.4.0] - 2026-04-02
### ✨ Features
- **Bundled Library:** Библиотека `LiquidCrystal_I2C` теперь входит в репозиторий (не нужно искать и качать отдельно).
- **Setup Script:** Добавлен `start.sh` для настройки в один клик (установка Python, зависимостей и запуск).

## [v1.3.0] - 2026-04-02
### ✨ Features
- **Smart Install:** `sender.py` теперь сам устанавливает `pyserial` и `psutil`, если их нет.
- **Requirements:** Добавлен файл `requirements.txt`.

## [v1.2.0] - 2026-04-02
### 🛠 Fixes
- **Reliability:** Исправлены критические ошибки стабильности соединения.
- **Metrics:** Добавлен фоллбэк для CPU/RAM через чтение `/proc`, если `psutil` недоступен.

## [v1.1.0] - 2026-04-02
### 🛠 Fixes
- Исправлен баг переполнения при 100% нагрузке.
- Удалены неиспользуемые импорты.

## [v1.0.0] - 2026-04-01
### 🚀 Initial Release
- Первый стабильный релиз PC-монитора для Arduino Nano + LCD 1602.
- Вывод шкал CPU и RAM.
- Передача данных через USB Serial.
